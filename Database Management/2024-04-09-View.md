1. CREATE VIEW
```SQL
CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;
```

2. ALTER VIEW
```SQL
ALTER VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;
```

3. DROP VIEW
```SQL
DROP VIEW view_name;
```

4. SELECT FROM VIEW
```SQL
SELECT * FROM view_name;
```
